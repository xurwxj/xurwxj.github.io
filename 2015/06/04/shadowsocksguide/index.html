shadowsocks服务器端ubuntu及客户端win7安装配置指南

2015-06-04

生活在墙内，但从事互联网，你不得不在必须的时候进行翻墙，这几年一直使用Linux和Mac，所以ssh及sshproxy用来翻墙是很方便的，但凑巧现在运营的同事是用win的，也需要翻墙，Google后发现了shadowsocks.

好东西，我们有一台墙外的vps，刚好可以给需要的人装一个服务器端，然后就可以顺溜翻墙了。

服务器端安装

  • apt-get install python-pip python-m2crypto
  • pip install shadowsocks

  • 配置文件：json格式，port_password部分是可以增加多个用户的，不同的端口不同的密码给不同的人用

  • ufw allow 8388/tcp 如果给多人用不同端口的，别忘记ufw也要开端口

  • vim /etc/init.d/shadowsocks
  • chmod +x /etc/init.d/shadowsocks
  • update-rc.d shadowsocks defaults
  • service shadowsocks reload

win7客户端安装

  • 到http://sourceforge.net/projects/shadowsocksgui/files/dist/ 下载客户端，win8要选择带dotnet的版本
  • 下载解压exe文件到你的程序目录（或你想要它去的目录）
  • 双击exe文件
